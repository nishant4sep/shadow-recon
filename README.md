\# Shadow Recon 🕶️



Automated reconnaissance toolkit for bug bounty \& security research.



A Bash-based pipeline that performs subdomain enumeration, live host probing, content extraction, and vulnerability pattern discovery using industry-standard tools.



\---



\## 🚀 Features



\- Subdomain Enumeration (Subfinder + Assetfinder)

\- Live Host Detection (httprobe)

\- Wayback URL Collection (optional)

\- HTTP Data Extraction (fff)

\- CORS Misconfiguration Detection

\- Server Fingerprinting

\- JavaScript File Discovery

\- Sensitive Data Detection using GF patterns:

&#x20; - API Keys

&#x20; - AWS Keys

&#x20; - JWT Tokens

&#x20; - LFI / RCE / SQLi / Redirects

\- Organized Output for easy analysis
